# Insurance Premium Prediction:


## Problem Statements:
The Goal of this Project is to give people an estimate of how much they need based on their individual health stituation. After that, customers can work with any health insurance carrier and its plans and perks while keeping the projected cost from our study in mind. This can assist a person in concentrating on the health side of an insurance policy rather than ineffective part.

## Step-by-Step Process:
* Understand the Problem Statement,
* Getting the knowledge about Domain,
* Collect & understand the data,
* Find the data distribution, Outliers, Collinearity, etc.,
* Create a Dashboard using Power BI,
* Create ML-Pipline to build a ML Model.
  * load and save data,
  * handle the missing values, outliers, apply mean-encoding, etc.,
  * split the data into train & test,
  * create ml model based on train dataset,
  * evaluate the model based on test data,
  * finalize the best ml model,
  * create web app to prediction based on new test data.
